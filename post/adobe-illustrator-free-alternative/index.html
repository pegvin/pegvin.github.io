Adobe Illustrator Free Alternative

Thursday, 24 October 2021

So you want to make some illustrations and you watched some tutorials where they are using “Adobe Illustrator” which is a paid software, and now you want a free alternative of it?

Inkscape is the answer

What is Inkscape?

Inkscape is an Open Source vector graphics editor, with capabilities similar to Adobe Illustrator, now it depends from person to person, but inkscape is has much better features and is beginner friendly.

Inkscape also uses very less resources of your computer, meaning you can literally run inkscape on a 2gb ram, intel dual core processor.

It supports many advanced SVG features (markers, clones, alpha blending, etc.) and great care is taken in designing a streamlined interface. It is very easy to edit nodes, perform complex path operations, trace bitmaps and much more. We also aim to maintain a thriving user and developer community by using open, community-oriented development.

Installation

  • Install On Windows

To install Inkscape on windows, goto the Inkscape’s Download Page and Click on the “Windows” Download button

And then select your CPU Architecture

if you don’t know your CPU Architecture then simple open “Command prompt” from the start menu, and run this command wmic os get osarchitecture and you’ll get your architecture.

  • Install On Linux

For Ubuntu You can simple open your terminal and run these commands:

sudo add-apt-repository ppa:inkscape.dev/stable
sudo apt update
sudo apt install inkscape

For Other Linux Distributions you can simple goto the Inkscape’s Download Page and Click on the “GNU/Linux” Download button and Download the “Appimage” file and open your terminal in the same directory where you have downloaded your “Appimage” file.

And Run this command chmod u+x <AppImage File> and replace the “<AppImage File>” with the filename of your downloaded file.

Now to run inkscape you can either run this Command ./<AppImage File> where “<AppImage File>” is the filename of your downloaded file, or double click on the file.
